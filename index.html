Eye in Mouth

Experience | Audio Visualizer

Is music just sounds? Can music be visual? What is the interplay between vision and sound; specifically, what is the interplay between the human organ that makes sounds and the organ that lets us observe the world. Experience the Eye in Mouth Music Visualizer!

Instructions



How to build

  • Download Zip file of all the files required here.
  • Unzip, open terminal on Mac and go into the folder musicineye
  • Run the command "make osx" on a Mac
  • After build is complete run the command "./VisualSine" to run the visualizer
  • Follow prompts to use the visualizer



Assignment Notes

Part 1 was easy, not much time was spent there. Getting the RFFT to work was the most challenging part of this project as I did not know what RFFT was before. It took a while understanding how windows worked and conceptualizing the conversion of time series data to frequency series data. However, sndpeek was really helpful in this part of the project. The rest of the project went smoothly but slowly. I wish I had more experience with OpenGL, I wouldn't have loved to implement some more visuals that were out of reach for this assignment.
